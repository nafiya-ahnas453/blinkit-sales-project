# blinkit-sales-project
##📊 Blinkit Sales Analysis Dashboard

Blinkit Retail Sales Performance Analysis

📖 Project Overview

This project is a Power BI dashboard created to analyze and visualize the sales performance of Blinkit (India’s last-minute delivery app).

The dashboard provides insights into:

Total Sales
Average Sales
Number of Items
Average Rating
Outlet performance
Item type contribution
Fat content distribution
Location and size-based sales analysis

The goal of this project is to help stakeholders understand business performance and make data-driven decisions.

🎯 Objectives
Analyze total revenue and average sales performance
Identify top-performing outlet types and locations
Compare sales across outlet sizes (Small, Medium, High)
Analyze product categories (Item Type)
Evaluate customer satisfaction using average rating
Understand the impact of fat content on total sales

📈 Key Insights
Tier 3 outlets generate the highest revenue.
Medium-sized outlets contribute the largest sales share.
Regular fat products generate more revenue than low-fat products.
Supermarket Type 1 shows the highest sales among outlet types.
Sales peaked around the 2018–2020 period.
